# Regis Employeee Prism Case Uses

This is a set of use cases for the Regis Employee Application using Prism API endpoints. Showing which endpoints are used for each use case and how they interact with the application.

## DTOs
You will find the DTOs used in the application for deserialization with Prism API endpoints.

## Prism
Prism API URL: `https://api.prismhr.com/prismhr-api/services/rest`
Modules:
- Benefits
- Client Master
- Code Files
- Employee
- Payroll

## Prism (api-1.30)
Prism API URL: `https://api.prismhr.com/api-1.30/services/rest`
Modules:
- Benefits
